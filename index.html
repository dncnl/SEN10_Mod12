Yale School of Art Website

A study in intentionally “ugly” web design and minimalism.

Why the Design Looks “Ugly” (On Purpose)

The Yale School of Art’s official website is a famous example of brutalist web design...

  • Brutalist Aesthetic: Harsh contrasts and minimal alignment.
  • Collaborative Editing: Any member can edit content.
  • Philosophical Expression: Rebellion against polished UI.
  • Low-Tech Authenticity: Built with basic HTML and CSS.

How It Could Be Improved

  • Consistent color palette for readability.
  • Clear typography and hierarchy.
  • Proper spacing and grid layout.
  • Responsive design across devices.
  • Subtle animations for polish.

Despite its “ugly” interface, the website is a bold artistic statement...
